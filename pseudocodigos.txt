PSEUDOCÓDIGOS Y NOTAS
----------------------RETO 1------------------------------------
Reto: Entrada array de números que pueden estar duplicados y sin ordenar; Salida array de números sin duplicar y ordenados.

Entrada: array con números
Salida: array con números

PSEUDOCODIGO:
preparegifts(gifts){
    
    setsinduplicados = Set(gifts)
    gifts = Array.from(setsinduplicados).sort(function(a, b){return a - b})
    return gifts
    
}

----------------------RETO 2------------------------------------
Reto: dado un número x de nombres, crear un marco adaptado al más largo que los contenga a todos
Entrada: Array de Strings
Salida: String

IMPORTANTE: NO SABEMOS SI EL ARRAY ESTÁ VACÍO QUÉ HABRÍA QUE DEVOLVER

PSEUDOCÓDIGO
createFrame(names){

    String marco
    int lengthMax = names[0].length

    //para establecer el nombre mas largo
    bucle(i = 1 < names.length){ //recorre names
        if(names[i].length > lengthMax){
            lengthMax = names[i].length
        }
    }

    //empieza el marco
    marco += '*'.repeat(lengthMax + 4) + '\n'

    bucle(i=0 < names.length){ //recorre names
        marco += '* ' + names[i] + ' '.repeat(lengthMax-names[i].length) + '*\n'
    }

    marco += '*'.repeat(lengthMax + 4) + '\n'

    return marco

}
